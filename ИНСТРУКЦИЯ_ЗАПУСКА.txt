ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА

1) ПЕРВИЧНАЯ НАСТРОЙКА (ОДИН РАЗ)
- Установите Node.js LTS: https://nodejs.org
- Откройте PowerShell в папке проекта:
  C:\Users\dimasik\Desktop\Создание презентации 3D-графики
- Установите зависимости:
  npm install

Если в PowerShell появляется ошибка про npm.ps1 (ExecutionPolicy), выполните:
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
Подтвердите: Y

2) ЗАПУСК САЙТА ДЛЯ РАЗРАБОТКИ
- В папке проекта выполните:
  npm run dev
- После запуска откройте адрес из терминала, обычно:
  http://localhost:5173/

3) КАК ЗАПУСКАТЬ САЙТ В ПОСЛЕДУЮЩЕМ
- Каждый раз после перезагрузки ПК:
  1. Откройте терминал в папке проекта
  2. Выполните:
     npm run dev
  3. Откройте в браузере:
     http://localhost:5173/

Примечание:
- npm install повторно делать не нужно, только если изменился package.json
  или вы удалили папку node_modules.

4) ПРОВЕРКА PROD-СБОРКИ (ПО ЖЕЛАНИЮ)
- Собрать проект:
  npm run build
- Предпросмотр production-сборки:
  npm run preview

Если порт 5173 занят, Vite автоматически предложит другой (например 5174).
