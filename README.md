# CodeKids

Образовательная платформа для обучения программированию с системой курсов, классов, заданий, чатов, онлайн-компилятора и геймификации.

## Стек

- **Frontend:** React 19 + Vite + Tailwind CSS + Radix UI
- **Backend API:** Express.js (Node.js 22)
- **База данных:** PostgreSQL 16
- **Компилятор:** JavaScript (Node.js vm) + Python (Pyodide/WebAssembly)

---

## Быстрый старт через Docker (рекомендуется)

### Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) — единственная зависимость

### Запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Snowxdxaxd/Xak.git
cd Xak

# 2. Создать .env (можно оставить по умолчанию для локального запуска)
cp .env.example .env

# 3. Собрать и запустить все сервисы
docker compose up --build
```

Откройте: **http://localhost:4000**

> При первом запуске Docker автоматически:
> - Поднимет PostgreSQL и применит схему БД (`server/schema.sql`)
> - Соберёт frontend (Vite build)
> - Запустит Express сервер, который раздаёт API и статику
> - **Создаст демо-аккаунты и тестовые данные** (см. раздел ниже)

### Остановка

```bash
docker compose down        # остановить контейнеры (данные сохранятся)
docker compose down -v     # остановить + полностью удалить данные БД
```

> После `docker compose down -v` и повторного `docker compose up --build` все демо-данные будут пересозданы автоматически.

## Локальный запуск (для разработки)

### Требования

- [Node.js LTS](https://nodejs.org) (v20+)
- [PostgreSQL 16+](https://www.postgresql.org/download/)

### Шаги

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Snowxdxaxd/Xak.git
cd Xak

# 2. Установить зависимости
npm install

# 3. Настроить переменные окружения
cp .env.example .env
# Отредактировать .env: указать DATABASE_URL с вашим паролем PostgreSQL

# 4. Создать базу данных
createdb -U postgres codekids

# 5. Применить схему
npm run db:init

# 6. Запустить frontend + backend одновременно
npm run dev:full
```

Откройте: **http://localhost:5173**

> Демо-аккаунты создаются автоматически при первом старте сервера.

### Команды

| Команда | Описание |
|---|---|
| `npm run dev:full` | Frontend (5173) + Backend (4000) вместе |
| `npm run dev:client` | Только Vite dev-сервер |
| `npm run dev:server` | Только Express API |
| `npm run build` | Production сборка frontend |
| `npm run db:init` | Инициализация схемы БД |

## Переменные окружения (`.env`)

| Переменная | По умолчанию | Описание |
|---|---|---|
| `DATABASE_URL` | `postgresql://postgres:postgres@localhost:5432/codekids` | Строка подключения к PostgreSQL |
| `API_PORT` | `4000` | Порт Express сервера |
| `JWT_SECRET` | — | Секрет для подписи JWT токенов (обязательно сменить в проде) |
| `ROOT_ADMIN_EMAIL` | `admin@codekids.local` | Email суперадмина |
| `ROOT_ADMIN_PASSWORD` | `Admin12345!` | Пароль суперадмина |
| `SMTP_HOST` | — | SMTP-сервер для отправки писем (опционально) |
| `SMTP_PORT` | `587` | Порт SMTP |
| `SMTP_USER` | — | Логин SMTP |
| `SMTP_PASS` | — | Пароль SMTP |
| `SMTP_FROM` | — | Адрес отправителя |

> Если SMTP не настроен, коды верификации при регистрации будут выводиться в консоль сервера.

## Основные возможности

- **Курсы** — публичные (для всех) и индивидуальные (🔒 только для записанных учеников)
- **Классы** — преподаватель создаёт класс, добавляет учеников по email, назначает курсы; ученики получают доступ автоматически
- **Онлайн-компилятор** — пишите и запускайте JavaScript и Python прямо в браузере
- **Чат** — мессенджер между пользователями платформы
- **Дашборд преподавателя** — статистика по курсам, ученикам, группам, динамика активности
- **Email-верификация** — подтверждение почты при регистрации через код
- **Геймификация** — уровни, баллы, достижения для учеников
