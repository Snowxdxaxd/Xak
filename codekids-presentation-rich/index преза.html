CodeKidsОбразовательная платформа для обучения программированию

Название команды: «Rich»

Курсы, классы, компилятор в браузере, видеоуроки, чат и геймификация — в одном веб-приложении.

Для кого эта платформа

Один продукт закрывает сценарии школы и семьи

Школы

Классы, расписание встреч, закрытые курсы для группы и отчётность по активности.

Преподаватели

Дашборд, задания с проверкой, онлайн-урок в том же интерфейсе, что и материалы.

Ученики

Понятный путь от темы к коду, геймификация и честный рейтинг среди группы.

Родители

Видимость прогресса без лишнего доступа к учебному процессу — доверие и прозрачность.

Технические решения · слайд 1 из 2

Разработка обязательного функционала

Базовый функционал, без которого платформа не выполняет свою учебную задачу.

Авторизация и регистрация

Простая процедура входа; сохранение прогресса, достижений и привязки к классу. JWT, роли, верификация email (SMTP при необходимости).

Уроки и модули

Последовательные учебные модули по темам: переменные, циклы, условия, функции и др. Структура курсов и уроков в PostgreSQL.

Уровни, XP, достижения

Система мотивации: очки опыта, уровни, награды, серии занятий — данные синхронизируются с профилем ученика.

Практические задания

Мини-проекты и задачи внутри приложения: написание скриптов, отправка решений, ручная и автоматическая проверка.

Тестирование знаний

Короткие викторины и экзамены для закрепления материала; результаты учитываются в прогрессе и при необходимости — в рейтинге.

Технические решения · слайд 2 из 2

Разработка дополнительного функционала

Заявленные возможности плюс усиление продукта функциями, которые редко совмещаются у конкурентов в одной системе.

Родительский контроль

Роль родителя: просмотр прогресса привязанных детей без доступа к коду заданий — прозрачность для семьи.

Сообщество и рейтинги

Внутренний мессенджер, таблица лидеров, сравнение успехов; общение без ухода на внешние мессенджеры.

Чего часто нет у конкурентов вместе: встроенный компилятор Python и JavaScript в браузере (без установки IDE); онлайн-уроки через WebRTC (Jitsi) прямо в интерфейсе; классы «как в школе» и индивидуальные курсы с закрытым доступом; развёртывание через Docker на своём сервере с полным контролем данных; двуязычный интерфейс (RU/EN); дашборд преподавателя с аналитикой по курсам и ученикам.

Уникальность и особенности · 1 / 2

Единая среда: от теории до кода и видео

Ученик не переключается между хостингом курсов, IDE, Zoom и чатом — теория, практика, звонок преподавателю и обсуждение в одном окне браузера.

  • Компилятор в продукте — запуск кода без установки окружения на домашнем ПК
  • Видеоуроки — встречи на базе Jitsi, расписание и контекст класса в одной системе
  • Гибкая проверка — и автотесты, и ручная оценка преподавателя для творческих задач

Уникальность и особенности · 2 / 2

Школа и родитель — одна модель доступа

Платформа заточена под реальный класс: публичные курсы, курсы только для группы, индивидуальные траектории и прозрачность для родителя.

  • Классы — преподаватель ведёт набор, назначает курсы, видит активность
  • Геймификация — XP, уровни, серии дней, лидерборд как дополнительная мотивация к оценкам
  • Open Source и Docker — школы и кружки могут развернуть копию у себя и дорабатывать под программу

Конкурентные преимущества

Сводка: сочетание функций, которое у массовых MOOC и LMS обычно разбросано по разным сервисам.

Критерий CodeKids (команда «Rich») Типично у других
LMS + классы + родитель в одном продукте часто только учитель/ученик
Код в браузере + задания Python / JS отдельная песочница или десктоп
Видеоуроки встроены (Jitsi) внешняя ссылка, отдельное приложение
Свой сервер и данные Docker только облако SaaS
Геймификация + чат + рейтинг встроено не всё вместе

Перспективы развития · 1 / 3

Обучение и контент

Развитие методической базы и библиотеки курсов под разные возрасты и уровни.

  • Новые модули по языкам и направлениям (веб, SQL, алгоритмы)
  • Банк тестов и автопроверяемых задач с разными уровнями сложности
  • Шаблоны уроков и рекомендованные траектории для преподавателей

Перспективы развития · 2 / 3

Продукт и пользовательский опыт

Улучшение интерфейса, аналитики и доступности для масштабирования аудитории.

  • Расширенная аналитика для школ: вовлечённость, слабые темы, отчёты для родителей
  • Улучшение мобильной вёрстки и PWA для учёбы с планшета
  • Развитие компилятора и песочницы (лимиты, безопасность, больше языков)

Перспективы развития · 3 / 3

Инфраструктура и экосистема

Надёжность, интеграции и сообщество вокруг открытого кода.

  • Production-hardening: SMTP, бэкапы БД, секреты и мониторинг
  • Интеграции с школьными системами (экспорт оценок, единый вход — по запросу заказчика)
  • Развитие open-source сообщества: форки под региональные программы и кружки программирования

Архитектура и стек (дополнительно)

Современный стек для быстрой разработки и предсказуемого развёртывания.
