ИНСТРУКЦИЯ: POSTGRES + ЗАПУСК + GITHUB

1. Установите Node.js LTS и PostgreSQL 16.
2. Создайте БД:
   createdb -U postgres codekids
3. Примените схему:
   psql -U postgres -d codekids -f server/schema.sql
4. Настройте .env (на основе .env.example).
5. Установите зависимости:
   npm install
6. Запустите backend и frontend:
   npm run dev:full
7. Откройте сайт:
   http://localhost:5173
8. Перед публикацией:
   npm run build

Для GitHub:
1. git init
2. git add .
3. git commit -m "Setup local PostgreSQL backend"
4. Создайте удаленный репозиторий и:
   git remote add origin <URL>
   git branch -M main
   git push -u origin main

ВАЖНО:
- Не коммитьте файл .env
- Коммитьте .env.example
