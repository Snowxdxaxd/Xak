CodeKidsПрограммирование для школьников в одной среде

Курсы, классы, онлайн-компилятор, видеоуроки, чат и геймификация — единое веб-приложение с Docker-развёртыванием.

Задача проекта

Объединить материалы, практику кода, коммуникацию учителя и ученика и контроль прогресса без набора разрозненных сервисов.

  • Снизить порог входа для преподавателя и ученика
  • Дать безопасную среду для запуска Python и JavaScript в браузере
  • Поддержать очный и дистанционный формат (видеозвонки Jitsi)

Что такое CodeKids

Полнофункциональная LMS с акцентом на программирование: не только контент, но и выполнение кода, задания с проверкой и социальные функции.

Курсы и классы

Публичные курсы и курсы только для класса.

Видеозвонки

Компилятор

Геймификация

XP, уровни, серии, таблица лидеров.

Стек: клиент

Современный SPA с дизайн-системой в духе shadcn/ui.

React 19 Vite TypeScript Tailwind CSS Radix UI

  • Адаптивный интерфейс, светлая и тёмная тема
  • Локализация (русский / английский)
  • Анимации интерфейса (Motion)

Стек: сервер и данные

Один backend раздаёт REST API и собранный фронтенд.

Node.js 22 Express.js PostgreSQL 16 JWT

  • Схема БД: пользователи, курсы, классы, заявки, встречи, чаты, уведомления
  • Миграции совместимости и сиды для демо-данных
  • Опциональная SMTP-верификация email

Развёртывание

Docker Compose поднимает приложение и базу; первый запуск готовит демо-аккаунты и курсы.

Один порт

Пользователь открывает http://localhost:4000 — и API, и SPA.

Повторяемость

Одинаковое окружение на машине преподавателя, ученика и сервера.

  • Healthcheck /healthz для мониторинга

Преимущества (1/2)

Почему это не «ещё один сайт с уроками».

  • Полный цикл — от теории до запуска кода и оценки в одной системе
  • Школьный контекст — классы, индивидуальные курсы, роль родителя
  • Видео встроено — не нужно отдельно договариваться о Zoom для урока
  • Открытый стек — можно развивать и кастомизировать под школу/кружок

Преимущества (2/2)

  • Геймификация — мотивация через XP и лидерборд, а не только оценки
  • Готовность к продакшену — роли, JWT, PostgreSQL, Docker
  • Мультиязычный UI — удобно для смешанных групп и отчётности
  • Гибкость проверки — авто- и ручная проверка заданий под разные типы задач

Сравнение с типичными аналогами

Упрощённо: где сильна CodeKids как связка функций.

Критерий CodeKids Часто у других
LMS + классы + родитель да часто только учитель/ученик
Компилятор JS/Python в продукте да редко вместе с классами
Видеоуроки (WebRTC) Jitsi внешняя ссылка или отдельное ПО
Геймификация + чат встроено не всегда в одном месте
Самостоятельный деплой Docker SaaS без своего сервера
