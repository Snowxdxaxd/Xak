НАСТРОЙКА БАЗЫ ДАННЫХ (SUPABASE)

1) СОЗДАНИЕ ПРОЕКТА
- Откройте https://supabase.com
- Создайте новый проект
- Сохраните:
  - Project URL
  - anon public key
  - service_role key (для edge function)

2) СОЗДАНИЕ ТАБЛИЦЫ KV
Вариант через SQL Editor (самый простой):
- Откройте SQL Editor в Supabase Dashboard
- Выполните SQL из файла:
  supabase/schema.sql

3) ПЕРЕМЕННЫЕ ДЛЯ ФРОНТЕНДА
- Скопируйте файл:
  .env.example -> .env
- Заполните:
  VITE_SUPABASE_URL
  VITE_SUPABASE_ANON_KEY
  VITE_API_BASE_URL

4) УСТАНОВКА SUPABASE CLI (по желанию)
Команда:
  npm i -g supabase

Проверка:
  supabase --version

5) ДЕПЛОЙ EDGE FUNCTION (если нужен свой backend)
- Логин:
  supabase login
- Линк на проект:
  supabase link --project-ref YOUR_PROJECT_ID
- Задать секреты:
  supabase secrets set SUPABASE_URL=https://YOUR_PROJECT_ID.supabase.co
  supabase secrets set SUPABASE_SERVICE_ROLE_KEY=YOUR_SERVICE_ROLE_KEY
- Деплой функции:
  supabase functions deploy server

6) ЗАПУСК САЙТА
  npm install
  npm run dev

7) ПРОВЕРКА API
- Должны открываться без ERR_FAILED:
  GET {VITE_API_BASE_URL}/courses
- Если ошибка осталась:
  - проверьте, что URL функции корректный
  - проверьте, что функция задеплоена
  - проверьте секреты SUPABASE_URL и SUPABASE_SERVICE_ROLE_KEY
